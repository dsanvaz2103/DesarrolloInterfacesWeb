Creación de la World Wide Web: (1989)


Tim Berners-Lee inventó la WWW, junto con el primer servidor web y el primer navegador, que funcionaban con el lenguaje HTML. 

Primer sitio web y lanzamiento público: (1991)

El primer sitio web se lanzó en el CERN, y la WWW llegó al público en general un año después. 

Auge de los navegadores gráficos: (1993)

La introducción de navegadores como Mosaic hizo la web más accesible y popular, permitiendo la visualización de contenido gráfico y la conexión de gobiernos, universidades y empresas. 

HTML 2.0: 1992

Creación y implementación de Tablas y Listas

JavaScript: (1995)

La aparición de JavaScript agregó dinamismo y interactividad a las páginas web. 

Fundación de Google: (1998)

El lanzamiento de Google revolucionó la forma en que se accedía a la información, consolidándose rápidamente como el buscador dominante. 

Web 2.0 : (2004)

El inicio de las redes sociales como Facebook marcó una nueva era, transformando la interacción entre usuarios y la forma de relacionarse en línea. 

HTML5 : (2008)

Esta nueva versión del lenguaje de marcado estándar sentó las bases para el desarrollo web moderno, añadiendo más funcionalidades y capacidades de diseño.
